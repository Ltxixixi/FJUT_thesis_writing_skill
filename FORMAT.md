# 论文格式详细规范

## A1 中文摘要格式

```
论文题目（居中、小2仿宋）

中文摘要（居中，4号黑体）

摘妥内容（300-800字，小4宋体，行距20pt）

关键词：关键词1；关键词2；关键词3（小4宋体，行距16pt）
```

## A2 英文摘要格式

```
ENGLISH TITLE（小3 Arial Black，行距24pt）

Abstract（4号Arial Black，行距24pt）

Content...（小4 Times New Roman，行距16pt）

Key words: word1; word2; word3（小4 Arial Black，行距16pt）
```

## A5 参考文献格式

### 5.1 基本要求

- 文献作者姓名写到第三位，余者写"等"或"et al."
- 引用标识置于所引内容末尾右上角，用方括号
- 不得将引用标识置于各级标题上
- 同一篇参考文献应用同一个序号标示

### 5.2 格式模板

**专著、论文集、学位论文、报告**：
```
[序号] 主要责任者.文献题名[文献类型].出版地：出版者，出版年：起止页码.
例：[1] 刘国，陈绍业，王风翥.图书馆目录[M].北京：高等教育出版社，1957：15-18.
```

**期刊文章**：
```
[序号] 主要责任者.文献题名[J].刊名，年，卷（期）：起止页码.
例：[2] 沈镭.青藏高原资源与发展互动关系探讨[J].自然资源学报，2000，15（4）:297-304.
```

**学位论文**：
```
[序号] 主要责任者.题名[D].保存地：保存单位，年份.
例：[3] 张筑生.微分半动力系统的不变集[D].北京：北京大学数学系数学研究所，1983.
```

**论文集中的析出文献**：
```
[序号] 析出文献主要责任者.析出文献题名[C]//原文献主要责任者.原文献题名.出版地：出版者，出版年：起止页码.
例：[4] 钟文发.非线性规划在可燃毒物配置中的应用[C]//赵玮.运筹学的理论与应用--中国运筹学会第五届大会论文集.西安：西安电子科技大学出版社，1996:468-471.
```

**报纸文章**：
```
[序号] 主要责任者.文献题名[N].报纸名，出版日期（版次）.
例：[5] 谢希德.创造学习的新思路[N].人民日报，1998-12-25（10）.
```

**国际、国家标准**：
```
[序号] 标准编号，标准名称[S].
例：[6] GB/T 16159-1996，汉语拼音正词法基本规则[S].
```

**电子文献**：
```
[序号] 主要责任者.电子文献题名[电子文献类型].电子文献出处或地址，发表日期/引用日期.
例：[7] 王明亮.关于中国学术期刊标准化数据库系统工程的进展[EB/OL].http://www.cajcd.edu.cn/pub/wml.txt/980810-2.html,1998-08-16/1998-10-04.
```

**未定义类型文献**：
```
[序号] 主要责任者.文献题名[Z].出版地：出版者，出版年.
例：[8] 中国航空规划设计研究院总院有限公司.工业与民用供配电手册[Z].北京：中国电力出版社，2017.
```

## A6 图表公式编号

- 图：图2-5（第2章第5个图）
- 表：表3-2（第3章第2个表）
- 公式：公式(5-1)（第5章第1个公式）
- 附注：用英文小写字母a、b、c...编号

## A7 三线表格式

所有表格均采用"三线表式"：
- 上下两粗线
- 表栏目与表内容间一细线
- 不用任何竖线

```
┌─────────────────────────────┐  ← 粗线（上下）
│      表头栏目      │   数据   │  ← 细线（栏目与内容间）
├─────────────────────────────┤
│      数据内容      │   数据   │
│      数据内容      │   数据   │
└─────────────────────────────┘  ← 粗线（上下）
```

续表格式：下页接排时左上角注明"续表*"，表头重复原栏内容。
