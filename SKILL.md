# 福建工程学院本科毕业设计（论文）写作规范

## 一、论文基本结构

### 1.1 必含章节（按顺序）

```
1. 封面（学校统一格式）
2. 中文题目（小2仿宋）
3. 摘要（含关键词3-8个）
4. Abstract（英文摘要，含Key words）
5. 目录
6. 正文（绪论 → 正论 → 结论）
7. 参考文献
8. 附录（如有）
9. 致谢
```

### 1.2 各部分字数要求

| 部分 | 字数要求 |
|------|----------|
| 中文摘要 | 300-800字 |
| 英文摘要 | 约400实词 |
| 设计说明 | 500-800字 |
| 结论 | 400-1000字 |
| 正文全文 | 8000字以上（工程类）或10000字以上（经管类） |

### 1.3 关键词要求

- 数量：3-8个（设计说明关键词一般5个左右）
- 排列：按词素外延层次从大到小
- 分隔符：用"；"
- 位置：摘要下方，另起一行

### 1.4 论文类型及字数要求

根据学科专业不同，选择对应的类型：

**计算机软件类**（本项目适用）：
- 独立完成一个足够工作量的软件模块
- 8000字以上的软件说明书
- 有完整的测试结果和参数指标
- 参考文献不低于10篇，外文文献2篇以上

**工程设计类**：8000字以上，参考文献10篇以上，外文2篇以上

**经济管理及人文社科类**：10000字以上，参考文献15篇以上，外文1-2篇

## 二、格式规范

### 2.1 用纸与页边距

- **纸张**：A4 (210mm × 297mm)
- **页边距**：
  - 上（天头）：25mm
  - 下（地脚）：20mm
  - 左（订口）：25mm
  - 右（切口）：20mm

### 2.2 字体规范

| 内容 | 中文字体 | 西文字体 | 字号 | 对齐 | 行距 |
|------|----------|----------|------|------|------|
| 论文题目 | 小2仿宋 | - | 小2号 | 居中 | 30pt |
| 摘要标题 | 黑体 | - | 4号 | 居中 | 24pt |
| 摘要正文 | 宋体 | - | 小4号 | - | 20pt |
| 关键词 | 宋体 | - | 小4号 | - | 16pt |
| 一级标题 | 黑体 | - | 小2号 | 居中 | 30-36pt |
| 二级标题 | 黑体 | - | 3号 | 顶格 | 18-24pt |
| 三级标题 | 黑体 | - | 小3号 | 顶格 | 12-15pt |
| 四级标题 | 黑体 | - | 小4号 | 顶格 | 6-9pt |
| 正文 | 宋体 | Times New Roman | 小4号 | - | 20pt |
| 表题 | 宋体 | - | 五号 | 居中 | - |
| 图题 | 宋体 | - | 五号 | 居中 | - |
| 参考文献 | 宋体 | - | 五号 | - | 17pt |
| 篇眉 | 宋体 | - | 五号 | 居中 | - |

### 2.3 摘要格式详解

**中文摘要**：
- 论文题目：居中、小2仿宋，行距30pt
- "摘  要"：居中、4号黑体，行距24pt
- 摘要正文：300-800字，小4宋体，行距20pt
- "关键词"：小4宋体，行距16pt，3-8个词用"；"分隔

**英文摘要**：
- ENGLISH TITLE：小3 Arial Black，行距24pt
- Abstract：4号 Arial Black，行距24pt
- 内容正文：小4 Times New Roman，行距16pt
- Key words：小4 Arial Black，行距16pt

### 2.4 标题层级编号

**理工类格式**（本项目适用）：
```
第1章 XXXX（居中，黑体小2号，段后30-36pt）
1.1 XXXX（顶格，黑体3号，段后18-24pt）
1.1.1 XXXX（顶格，黑体小3号，段后12-15pt）
1.1.1.1 XXXX（顶格，黑体小4号，段后6-9pt）
```

**文法经管类格式**：
```
一、XXXX（居中，黑体小2号）
（一）XXXX（顶格，黑体3号）
1. XXXX（顶格，黑体小3号）
（1）XXXX（顶格，黑体小4号）
```

**重要规则**：
- 层级不超过4级
- 每级末尾不加标点
- 各层标题均单独占行
- 每章应另起一页
- 标题字数一般不超过15字

**正文中的分项编号**：
- 总项分项：（1）、（2）、（3）...
- 分项中的小项：①、②、③...或数字加半括号

### 2.5 行距与段间距

- **正文行距**：固定行间距 20pt
- **段后间距**：
  - 大标题（一级标题）：30-36pt
  - 一级节标题（二级标题）：18-24pt
  - 二级节标题（三级标题）：12-15pt
  - 三级节标题（四级标题）：6-9pt
- **参考文献行距**：17pt，段前加3pt
- **参考文献标题间距**：30-36pt
- 注意：不要在一篇参考文献段落的中间换页

### 2.6 篇眉与页码

- **篇眉**：从第一章开始，宋体五号居中，下方加横线
  - 奇数页：`作者姓名 论文题目`
  - 偶数页：`福建工程学院本科生毕业设计（论文）`
- **页码**：从绪论开始，阿拉伯数字连续编排，底端居中

## 三、正文写作要求

### 3.1 绪论内容

1. 研究背景（国内外发展概况及存在的问题）
2. 研究目的和主导思想
3. 拟解决的主要问题
4. 研究方法、步骤
5. 研究意义或价值

### 3.2 正论结构

```
① 采用什么研究方法
② 详细表述研究过程
③ 取得了哪些研究结果
```

### 3.3 结论内容

- 归纳总结研究工作
- 与他人研究成果比较
- 指出成果价值和尚待解决的问题
- 提出深入研究方向和建议

### 3.4 语言规范

- 观点鲜明正确，具有创新性
- 论证周密、合乎逻辑
- 语言清晰流畅，避免感情色彩词语
- 使用规范的科学技术术语
- 标点符号按新闻出版署公布的"标点符号用法"正确使用

### 3.5 专业术语规范

- 科学技术术语尽量采用全国自然科学名词审定委员会公布的规范词
- 使用外文缩写代替专业术语时，首次出现应在括号内注释其全称
- 外国人名一般采用英文原名，按"名前姓后"原则书写

### 3.6 附录规范

- 理工类附录序号：附录1、附录2...
- 文法经管类附录序号：附录一、附录二...（用中文数字）

## 四、图表、公式与注释

### 4.1 图的要求

- **位置**：随文给出，先见文后见图（不许图先文后）
- 图序按章节编号（如：图2-5）
- 图名置于图下方居中
- 图中术语、符号、单位与正文一致
- 同一图上不同曲线用不同形状标出

### 4.2 表的要求

- **位置**：随文给出，先见文后见表（不许表先文后）
- **表题位置**：表的上方，居中排写
- 表序按章节编号（如：表3-2）
- 续表：下页接排时左上角注明"续表*"，表头重复

### 4.3 公式要求

- 居中书写
- 编号用括号括起，写在右边行末
- 序号按章节编号（如：公式(5-1)）
- 图、表、公式与正文间距适当

### 4.4 注释规范

- **理工类**：一般采用页下注
- **哲社类**：可采用页下注或篇末注
- **禁止**：原则上不采用页中注（夹在正文中的注）
- 页下注只写在注释符号出现的同页，不得隔页

## 五、参考文献规范

### 5.1 基本要求

- 引用近5年文献占比不低于40%
- 引用标识在正文末尾右上角，用方括号
- 统一用阿拉伯数字顺序编号
- 放在结论后，不得放在各章之后
- 文献作者姓名写到第三位，余者写"等"或"et al."
- 不得将引用标识置于各级标题上

### 5.2 格式模板

**专著**：
```
[序号] 主要责任者.文献题名[M].出版地：出版者，出版年：起止页码.
```

**期刊**：
```
[序号] 主要责任者.文献题名[J].刊名，年，卷（期）：起止页码.
```

**论文集**：
```
[序号] 析出文献主要责任者.析出文献题名[C]//原文献主要责任者.原文献题名.出版地：出版者，出版年：起止页码.
```

**报纸**：
```
[序号] 主要责任者.文献题名[N].报纸名，出版日期（版次）.
```

**标准**：
```
[序号] 标准编号，标准名称[S].
```

**电子文献**：
```
[序号] 主要责任者.电子文献题名[电子文献类型].地址，发表日期/引用日期.
```

## 六、写作检查清单

完成论文后，逐项核对：

**格式检查**：
- [ ] 题目简洁，不超过20字
- [ ] 中文摘要300-800字，关键词3-8个
- [ ] 英文摘要约400实词
- [ ] 标题层级不超过4级，末尾不加标点
- [ ] 页边距符合要求（上下25mm，左右20mm）
- [ ] 正文字体字号、行距符合规范
- [ ] 一级标题居中，其他顶格
- [ ] 参考文献行距17pt，段前3pt

**内容检查**：
- [ ] 参考文献引用格式正确
- [ ] 近5年文献占比≥40%
- [ ] 引用标识置于正文末尾右上角
- [ ] 引用标识不在标题上
- [ ] 图、表、公式编号分章连续

**其他**：
- [ ] 篇眉页码格式正确
- [ ] 字数满足8000字以上要求
- [ ] 外文文献不少于2篇
